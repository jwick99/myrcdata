myRCdata app

This app is designed to help keep control of all the data elements of all your RC planes.

When a plane is added you can add:

  • name
  • wingspan
  • motor
  • esc
  • battery
  • prop
  • size
  • battery
  • rates
  • notes

Besides keeping the details of each aircraft, the app allows you to track how many times you have flown the aircraft and the last date if was flown.
